# mime-type

Looks up the mime type of files.

[![Build Status](https://travis-ci.org/ph3nx/mime-type.svg)](https://travis-ci.org/ph3nx/mime-type)

# Installation
```
npm i [-g] mime-type
```

# Usage

1. require
```js
var mt = require('mime-type');
console.log(mt());
```

2. cli
```
$ mime-type file.ext
```
